Microsoft Excel consta de hojas para realizar cálculos. Una hoja de cálculo es como una hoja de contabilidad.
Con filas y columnas que se cruzan para formar celdas que guardan datos. Los datos pueden ser número o texto que se introducen; calculan los valores de las fórmulas basados en los otros números y también sirve como apoyo para algún trabajo ya que en ella podemos calcular cualquier tipo de operación
Las funciones, son fórmulas predefinidas formadas por unos valores llamados argumentos, qué forman parte una estructura u orden determinado, llamado sintaxis, y que sirven para realizar cálculos y darnos un resultado. Pueden ejecutar cálculos simples o completos
Como por ejemplo MAX.
Que es una de las funciones estadísticas de más utilizadas en Excel.
MAX representa las tres primeras letras de la palabra MAXIMO y se utiliza para encontrar el número más grande mayor o máximo en una lista de valores o argumentos ¿Cuáles son las funciones básicas de Excel?

Revisa a continuación las funciones básicas de Excel más utilizadas:
Suma

La función sumar en Excel permite sumar una fila o columna. Pero también se puede seleccionar la celda al final de una fila o columna de manera rápida, ¿cómo?
Selecciona la celda y haz clic en Ctrl + Shift + = para activar la función. Esto es equivalente a “Autosuma”.

RESTA

Se usa para restar datos de filas o columnas. Para esto debes usar el signo menos -.
Por ejemplo, si escribes la fórmula =10-5 en una celda, la celda mostrará 5 como resultado.

Multiplicar

Se usa para multiplicar dos matrices. Para realizar esta tarea se usa el * (asterisco).
Por ejemplo, si escribes =8*10 en una celda, la celda muestra como resultado 80.

Dividir
Para realizar esta tarea debes usar / (barra diagonal).
Por ejemplo, si escribes =10/5 en una celda, la celda mostrará 2.
*Ojo que es importante escribir un signo igual (=) en la celda antes de escribir los números y el operador /. De lo contrario, Excel interpretará estos datos como una fecha.

Promedio
Con esta puedes obtener la media de un conjunto de datos registrados en la hoja de cálculo.
Por ejemplo, si necesitas saber la media de ventas diarias de un negocio, registradas en el intervalo A1-A245, se utilizaría la expresión =PROMEDIO(A1:A245) y Excel arrojaría el monto diario que vende el negocio.
CTRL Z / CTRL Y
Sabemos que el Ctrl + Z nos sirve para revertir errores. Sin embargo, también existe otra opción que puedes usar para recuperar lo borrado: comando Ctrl + Y.
Para esto pulsa Ctrl + Y para revertir un “error” que pudiste eliminar con Ctrl + Z.

¿Qué es una función en Excel? "Para comenzar, hablemos del concepto clave: ¿Qué es una función?.
En Excel, una función es como una fórmula predefinida que realiza cálculos automáticamente. En lugar de hacer manualmente operaciones matemáticas o buscar información a través de cientos de filas de datos, puedes usar una función para hacerlo por ti.

Las funciones en Excel siguen un patrón muy simple: escribes el signo igual =, luego el nombre de la función y finalmente los argumentos entre paréntesis. Estos argumentos pueden ser números, rangos de celdas, texto o cualquier valor que necesites. Por ejemplo, si deseas sumar los valores de las celdas A1 a A5, escribirías =SUMA(A1:A5). Excel se encargará de sumar esos valores sin que tengas que hacerlo manualmente."

Explicación visual y ejemplos
"Veamos algunos ejemplos de funciones en acción. Aquí, en esta hoja de cálculo, tenemos una lista de números. Digamos que queremos sumar estos valores rápidamente, lo que puede ser útil si trabajamos con presupuestos, informes financieros o cualquier tipo de análisis numérico. Usamos la función SUMA, como puedes ver, Excel toma automáticamente el rango de celdas que seleccionamos y nos da el resultado.
Lo mismo ocurre con la función PROMEDIO, que calculará el promedio de los valores seleccionados. Esto es útil si quieres obtener un resumen rápido de un conjunto de datos, como el promedio de ventas o de rendimiento en un período de tiempo. Solo necesitamos escribir =PROMEDIO(A1:A5), y obtendremos el valor promedio de esas celdas."

SUMA: Cómo y cuándo usarla
"Ahora, vamos a explorar en profundidad la función SUMA. La función SUMA es una de las más utilizadas y sencillas de Excel, ya que te permite agregar valores rápidamente. ¿Cuándo deberías usarla? Cada vez que necesites sumar múltiples valores en un rango de celdas, ya sea en columnas o filas.
Por ejemplo, si tienes una lista de gastos mensuales y quieres saber el total, puedes seleccionar todo el rango de celdas donde están los valores de los gastos y usar =SUMA(A1:A12). Excel automáticamente sumará esos valores por ti. Imagina hacer esto manualmente, especialmente si tienes decenas o cientos de números; la función SUMA te ahorra tiempo y reduce los errores."

PROMEDIO: Aplicaciones prácticas
"Pasemos ahora a la función PROMEDIO. Esta función calcula el promedio de los números que le indiques. Es ideal si trabajas con promedios de notas, ventas, temperaturas, etc.
Por ejemplo, si tienes las ventas de tu empresa de lunes a viernes, puedes usar =PROMEDIO(A1:A5) para obtener el promedio diario. Esto es especialmente útil si tienes grandes volúmenes de datos, y necesitas un número representativo que te diga cómo estás desempeñándote en promedio. El uso de PROMEDIO es fundamental cuando se trata de analizar tendencias."

BUSCARV: Encuentra datos específicos
"A continuación, tenemos la función BUSCARV, una de las herramientas más poderosas para encontrar datos en hojas de cálculo grandes. Imagina que tienes una tabla con información de cientos de productos y necesitas encontrar rápidamente el precio de uno en particular. Con BUSCARV, puedes buscar un valor en la primera columna de una tabla y hacer que Excel te devuelva otro valor de la misma fila, pero en una columna diferente.
La estructura es =BUSCARV(valor_buscado, rango, índice_columna, [ordenado]). Aquí el valor buscado es lo que queremos encontrar, el rango es el área de la tabla donde buscar, el índice de columna es la columna que contiene el valor que quieres devolver, y el ordenado es opcional, pero normalmente es 'FALSO' para búsquedas exactas. Es muy útil para bases de datos, hojas de inventario, o cualquier situación donde necesites localizar información específica rápidamente."

SI: Toma decisiones en función de condiciones
"Finalmente, hablemos de la función SI, que es extremadamente flexible y nos permite hacer comparaciones lógicas. La estructura es =SI(prueba_lógica, valor_si_verdadero, valor_si_falso), lo que significa que puedes definir una condición, y Excel devolverá un valor si esa condición es verdadera, o uno diferente si es falsa. Por ejemplo, si tienes una lista de calificaciones y quieres saber si un estudiante aprobó o reprobó, puedes usar la función SI para automatizar esa tarea. Podrías escribir algo como =SI(A1>70, "Aprobado", "Reprobado"). Si la nota en A1 es mayor a 70, Excel mostrará 'Aprobado', de lo contrario, mostrará 'Reprobado'. La función SI es increíblemente versátil porque te permite automatizar decisiones basadas en los datos."
